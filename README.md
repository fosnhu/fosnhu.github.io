# FOSNHU Site

Furries of SNHU community website.

## Building this website

To run this project locally using Vite, follow these steps:

### Clone this repo
```shell
git clone https://github.com/FOSNHU/FOSNHU-Site.git
cd FOSNHU-Site
```

### Install dependencies
```shell
npm install
```

### Run the dev server
```shell
npm run dev
```

## Tools

### Make a new update
```shell
npm run new-update
```

### Build bundles for production
```shell
npm run build:bundles
```

## Deployment

The site is automatically deployed to GitHub Pages when pushing to the `main` branch.
